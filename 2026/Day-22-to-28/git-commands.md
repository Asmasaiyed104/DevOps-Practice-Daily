# Git Commands Reference

---

# Setup & Config

## git --version

Shows the installed Git version.

Example:

```bash
git --version
```

---

## git config --global user.name

Sets Git username.

Example:

```bash
git config --global user.name "Asma Saiyed"
```

---

## git config --global user.email

Sets Git email address.

Example:

```bash
git config --global user.email "asmasaiyed104@gmail.com"
```

---

## git config --list

Shows all Git configuration settings.

Example:

```bash
git config --list
```

---

# Repository Setup

## git init

Creates a new Git repository.

Example:

```bash
git init
```

---

## ls -la

Shows hidden files including `.git`.

Example:

```bash
ls -la
```

---

# Basic Workflow

## git status

Shows current repository status.

Example:

```bash
git status
```

---

## git add

Stages files for commit.

Example:

```bash
git add git-commands.md
```

---

## git add .

Stages all changed files.

Example:

```bash
git add .
```

---

## git commit

Creates a snapshot of staged changes.

Example:

```bash
git commit -m "Add Git notes"
```

---

# Viewing Changes

## git log

Shows detailed commit history.

Example:

```bash
git log
```

---

## git log --oneline

Shows compact commit history.

Example:

```bash
git log --oneline
```

---

## git diff

Shows file changes before commit.

Example:

```bash
git diff
```

---

# Help Commands

## git help

Shows Git help documentation.

Example:

```bash
git help
```

---

## git status --help

Shows help for git status command.

Example:

```bash
git status --help
```
