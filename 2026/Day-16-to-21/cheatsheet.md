# Day 19 Easy Human Cheat Sheet

## Folder Work

mkdir
= make new folder

ls
= see files and folders

cd
= go inside folder

pwd
= show current location

rmdir
= remove empty folder

Memory Trick:

mkdir → make directory
cd → change directory

---

# File Work

touch
= create new file

nano
= open file editor

cat
= show file content

chmod +x
= give execute permission

./file.sh
= run script

Memory Trick:

touch → touch new file
cat → show everything

---

# Log Rotation

find
= search files

gzip
= compress log file

mtime
= old file check

Memory Trick:

gzip → zip old logs

---

# Backup Commands

tar
= create backup archive

date
= current date and time

rm
= remove file

Memory Trick:

tar → pack files together

---

# Crontab

crontab -l
= show scheduled jobs

crontab -e
= edit cron jobs

Memory Trick:

cron = automatic timer

---

# Cron Timing

0 2 \* \* \*
= daily 2 AM

0 3 \* \* 0
= Sunday 3 AM

_/5 _ \* \* \*
= every 5 minutes

Memory Trick:

Minute Hour Day Month Week

---

# Monitoring Commands

df -h
= disk usage

free -h
= memory usage

uptime
= server running time

ps aux
= running processes

Memory Trick:

df → disk free
free → free memory

---

# Script Safety

set -e
= stop if error

set -u
= stop undefined variable

set -o pipefail
= fail if pipeline fails

Memory Trick:

e → error stop
u → undefined stop

---


Memory Trick:

add → prepare
commit → save
push → upload
